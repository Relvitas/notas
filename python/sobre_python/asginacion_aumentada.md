### Asignaciones aumentadas en Python

- `+=` → **Suma y asigna**
  
  `x += 3   # x = x + 3`

- `-=` → **Resta y asigna**
  
  `x -= 2   # x = x - 2`

- `*=` → **Multiplica y asigna**
  
  `x *= 4   # x = x * 4`

- `/=` → **Divide y asigna** (resultado float)
  
  `x /= 2`

- `//=` → **División entera y asigna**
  
  `x //= 2`

- `%=` → **Módulo y asigna** (resto de la división)
  
  `x %= 3`

- `**=` → **Potencia y asigna**
  
  `x **= 2`
